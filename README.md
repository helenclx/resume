# Helen Chong's Résumé

Helen Chong's résumé. Built with Eleventy

## Features

- Fully Customizable
- Semantic HTML
- Theming with TailwindCSS (Dark/Light)
- Accessible (WCAG AA)
- Self-Contained (no external resources)
- Search Engine Optimized (meta, JSON-LD, etc...)

## Run Locally

Requires Node.js.

To install the necessary packages, run this command in the root folder of the repository:
```
npm install
```

**Commands**
- Run `npm start` for a development server and live reloading
- Run `npm run build` to generate a production build
